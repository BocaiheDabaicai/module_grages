# xhsndl 模板库及功能库

#### 1. 配置条件

nodeJs：需要24以上的稳定版本

vue：支持vue3以上的稳定版本

vite：由vite构建

#### 2. 快速开始

- 拉取依赖包文件

```pnpm
pnpm i
```

- 启动项目

```tex
npm run dev
```

- 项目打包

```tex
npm run build
```
